🏋️‍♂️ Workout Manager - Versão Beta (V0.1)
Bem-vindo ao Workout Manager, uma solução moderna e intuitiva para a gestão de planos de treino, desenvolvida pela Larsson Technologies.

Este software foi concebido para simplificar a organização da rotina desportiva, oferecendo uma interface limpa e funcional para utilizadores que procuram otimizar a sua performance física.

🚀 Sobre o Projeto
O sistema está atualmente na sua fase Beta. Isto significa que o programa está 100% funcional no que toca à lógica de interface, navegação e processamento de dados em tempo real. É a base sólida sobre a qual construiremos o futuro da gestão fitness digital.

🛠️ Tecnologias Utilizadas
Linguagem: Python

Framework Web: Flask (Leve, rápido e escalável)

Frontend: HTML5, CSS3 (com foco em responsividade)

📋 Funcionalidades Atuais
Visualização completa da interface de gestão.

Criação dinâmica de rotinas de treino.

Navegação fluida entre categorias de exercícios.

Estrutura de backend pronta para processamento de variáveis de treino.

⚠️ Estado de Desenvolvimento & Próximos Passos
Como qualquer versão Beta, o foco atual foi a experiência do utilizador (UX) e a arquitetura do código.

Nota Importante: Atualmente, o programa ainda não realiza a persistência de dados (gravação em base de dados). As informações inseridas são processadas em memória, mas não são guardadas após o encerramento da sessão.

Roadmap de Melhorias:

Implementação de Base de Dados: Integração com SQLite ou PostgreSQL para gravação permanente.

Sistema de Login: Autenticação segura para múltiplos utilizadores.

Histórico de Progresso: Gráficos de evolução de cargas e medidas corporais.

Executa o ficheiro principal: python app.py

Acede no teu browser a: http://127.0.0.1:5000
